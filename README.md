# Hi there, I'm An Tran ^^

I'm a software developer from Vietnam. I grew up in Đà Lạt and moved to Sài Gòn for university.

I build end-to-end products, and almost every one of them started from a problem I lived through myself. Living alone got too expensive and I couldn't find a good way to look for roommates — so I built **roomieVerse**. I watched a café schedule its staff shifts by hand every single week — so I'm building **shiftbot**. I once froze instead of helping at a motorbike accident and carried that regret for years — so I made **Our Warmth**.

### How I work

I build with AI agents, but I don't hand it a prompt and ship whatever runs. I treat AI as a very fast typist, not an employee: I decide the use cases, design the flows, make the architecture calls, redo every screen until it *feels* right, and deploy everything myself. **I only ship what I can explain.**

### What I'm building right now

- **roomio** — rooming-house management SaaS · SvelteKit, Postgres, Telegram Mini App
- **shiftbot** — automatic shift scheduling for small cafés · Go, LLM parsing, a hand-written solver
- **roomieVerse** — find roommates & shared housing, broker-free · [roomieverse.blog](https://roomieverse.blog) (live, pre-launch)
